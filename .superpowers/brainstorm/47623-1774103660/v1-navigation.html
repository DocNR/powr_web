Which tabs ship in v1?

The current app has 8 navigation tabs. For v1, we need the minimum set that supports the end-to-end workout flow. Pick all that should ship.

1

Workout

The core flow: pick a template, track sets in real-time, complete & publish. This is non-negotiable for v1.

2

Library

Browse/search workout templates and exercises. Without this, users can't find templates to start a workout.

3

Log / History

View past workouts. Gives users a reason to come back — see their streak, recent sessions.

4

Profile

Nostr identity, settings, logout. Could be a settings sheet instead of a full tab.

5

Social

See others' workouts, follow athletes. Nice-to-have but not required for core workout tracking.

6

Progress

Charts, PRs, trends over time. High value but can launch post-v1 once there's data to show.

Debug and Home tabs omitted — Debug is dev-only, Home can be the default landing state of the Workout tab.
